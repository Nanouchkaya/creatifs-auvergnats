## 🚀 Quick start

Website develop with Gatsby for the association "Collectif Les Créatifs Auvergnats".

## 💫 What's inside?

A quick look at the technolgies use for it:

    - ReactJS (with hooks)
    - Gatsby*
    - ESLint
    - Prettier
    - Yarn
    - Node
    - GraphQL
    - Sass
    - CSS module
    - AirBnB Lottie

    * Gatsby plugins : 
    - React Helmet
    - gatsby-images & background-images
    - sharp & transformer-sharp
    - source-filesystem
    - manifest
    - sass
