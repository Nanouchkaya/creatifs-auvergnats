## 🚀 Quick start

Website develop with Gatsby for the association "Collectif Les Créatifs Auvergnats".

## 💫 What's inside?

A quick look at the technolgies use for it:

    - Gatsby* /ReactJS (with hooks)
    - Netlify
    - ContentfulCMS
    - GraphQL
    - ESLint
    - Prettier
    - Yarn
    - Sass
    - CSS module
    - AirBnB Lottie

    * Gatsby plugins : 
    - React Helmet
    - sharp & transformer-sharp
    - source-filesystem
    - manifest
    - sass
